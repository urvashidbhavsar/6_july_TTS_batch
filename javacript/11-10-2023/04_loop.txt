loop is a count number of iteration.

in js there is 6 type of loop :
(1) while loop 
(2) do.. while loop 
(3) for loop 
(4) for..of loop (array)
(5) for..in loop (object)
(6) foreach loop (both)

syntax :
(1) while loop 
while(condition){
    statement 
    increment / decrement 
}

(2) do..while loop 
do{
    statement 
    increment / decrement 
}while(condition)

(3) for loop 
for(declaration; condition; increment/decrement){
    statement
}
